# LingoCards

Structured web-based German verb flashcard editor with a modular layout engine and printable PDF export (A6 cards).  
Includes JSON-driven content management and optional AI panel integration (LM Studio).

## Features
- A6 flashcard layout editor (print-ready)
- JSON import/export for verb sets and cards
- PDF export with consistent preview-to-PDF rendering
- Modular state/boxes architecture (migrations-friendly)
- Optional AI tools integration (LM Studio)

## Tech
- JavaScript (web app)
- JSON-based data model
- PDF generation pipeline

## Status
Active development. Stable builds are tagged and released periodically.
