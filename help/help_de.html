Hilfe und Funktionen

LingoCard ist eine App zum Erstellen und Drucken von Karteikarten für deutsche Verben. Ziel: weniger Handarbeit, mehr saubere Karten und PDF ohne Überraschungen.

01 Was ist eine Karte?
Grundlage

Eine Karte ist ein Verb und die dazugehörigen Daten:

  • Übersetzungen und Kontext
  • Verbformen (Präsens / Präteritum / Partizip II / Hilfsverb)
  • Beispiele (DE/RU)
  • Rektion (Valenz), Tags, Häufigkeit

Eine Karte ist eine Struktur. Das Design kann sich ändern — die Bedeutung bleibt.

02 Linke und rechte Liste
Arbeitslogik

In der App gibt es zwei Listen.

Linke Liste — die Haupt-/Referenzliste. Meistens wird sie für Druck und Export verwendet.

Rechte Liste — Entwurf (Draft). Dort kann man testen, ergänzen und korrigieren, ohne die Hauptliste zu riskieren.

Idee: Der Entwurf ist da, damit ein Fehler nicht zur Katastrophe wird.
03 Karten erstellen und bearbeiten
Praxis

Karten können manuell erstellt und bearbeitet werden. Außerdem kann KI genutzt werden (wenn verbunden).

  • Erstellen Sie neue Karten zuerst im Entwurf, wenn Sie das Ergebnis prüfen möchten.
  • Verschieben Sie eine Karte in die Hauptliste, wenn sie korrekt aussieht.

Bester Ansatz: zuerst inhaltlich korrekt, danach das Layout feinjustieren.

04 Blöcke und Layout
Layout

Eine Karte besteht aus Blöcken (Übersetzung, Formen, Beispiele usw.).

Typischerweise kann man:

  • Blöcke ein-/ausblenden
  • verschieben und Größe ändern
  • das Layout an das Druckformat anpassen

Idee: Sie bauen die Karte wie einen Baukasten — passend zu Ihrem Format.

05 PDF-Export und Drucken
Druck

LingoCard kann PDF für den Druck erstellen.

  • Vorschau soll dem PDF entsprechen.
  • Wenn etwas nicht passt, sieht man das vor dem Export.

Tipp: Erst 1–2 Karten perfektionieren, dann massenhaft exportieren.

06 Lokale KI: LM Studio (Installation & Setup)
KI

LM Studio ist eine App zum lokalen Ausführen von Sprachmodellen auf Ihrem PC.

Offizielle Website: https://lmstudio.ai

Start:

  1. LM Studio von der offiziellen Website herunterladen und installieren.
  2. LM Studio starten und ein Modell herunterladen (ohne Modell läuft es nicht — normal).
  3. Bereich Local Server öffnen und den Server starten (Start Server).

Empfehlungen für ein Modell:

  • Typ: Chat / Instruct (nicht Base)
  • Größe: 7B–8B (Balance aus Qualität und Geschwindigkeit)
  • Format: GGUF
  • Quantisierung: Q4_K_M oder Q5_K_M

Mit LingoCard verbinden:

Im AI Control Panel den Endpoint eintragen:

http://localhost:1234/v1
Wichtig: Wenn LM Studio nicht läuft, funktioniert die App trotzdem — nur ohne KI. KI ist ein Helfer, kein Muss.

Wenn es langsam ist: kleineres Modell nutzen, schwere Apps schließen, prüfen ob der Server läuft.

07 Wie kann man Verben für die Generierung eingeben?
AI

Für die Generierung gibt es mehrere Eingabewege — von einem Infinitiv bis zu einer ganzen Liste.

1) Ein Infinitiv (Feld „Infinitiv“)

  • Infinitiv eingeben und Generieren drücken.
  • Autovervollständigung: Tab oder → übernimmt den Vorschlag.
Neuer Modus: du kannst das Verb auch auf Russisch oder auf Englisch eingeben. Die AI‑Panel versucht es (über LM Studio) in einen deutschen Infinitiv zu übersetzen und erzeugt danach die Karte wie gewohnt.

2) Mehrere Infinitive im selben Feld

Mehrere Verben sind möglich:

  • mit Komma: gehen, kommen, bleiben
  • mit Leerzeichen: gehen kommen bleiben
  • mehrteilige Verben in Anführungszeichen: "Rad fahren" "spazieren gehen"
Wichtig: Ein Leerzeichen zählt als Trenner. Für mehrteilige Verben bitte Anführungszeichen verwenden.

3) Liste per Datei („Verben aus Datei hinzufügen“)

Gut für Batch-Generierung.

  • Empfohlen: 1 Infinitiv pro Zeile.
  • Auch möglich: Komma/Leerzeichen — aber Zeilen sind zuverlässiger.

4) Karten per JSON importieren (ohne Generierung)

Wenn Sie fertige Karten als JSON haben, können Sie sie in die App importieren. Das ist kein KI-Generieren, sondern Datenimport.

  • Import nutzen, wenn die Karten bereits ausgefüllt sind.
  • Unvollständige Karten zuerst im rechten Entwurf verbessern und dann in die linke Liste übernehmen.

Empfohlener Ablauf

  1. Generieren → Karte erscheint in der rechten Liste
  2. Prüfen/verbessern → in die linke Liste übernehmen
  3. PDF exportieren
08 Datensicherheit & was tun, wenn es „komisch“ ist
ruhig bleiben

Wenn sich die App merkwürdig verhält:

  • nicht sofort alles löschen
  • prüfen, in welcher Liste Sie arbeiten (Hauptliste / Entwurf)
  • prüfen, ob ein Modus aktiv ist, der Elemente ausblendet/filtern kann

Beste Strategie: kleine Änderungen und häufige Kontrolle des Ergebnisses.
